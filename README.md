# Next.js Full-Stack Job Task

## Project Description
A simple Next.js 15/16 application featuring a landing page with 7 sections, mock authentication with cookies, item listing from an Express server, and protected route to add new items.

## Setup Instructions
1. **Clone the project**
2. **Backend Setup:**
   - Go to `server` folder: `cd server`
   - Install dependencies: `npm install`
   - Run server: `node index.js` (Runs on port 5000)
3. **Frontend Setup:**
   - Go to root folder: `cd scic-job-task`
   - Install dependencies: `npm install`
   - Run dev: `npm run dev` (Runs on port 3000)
